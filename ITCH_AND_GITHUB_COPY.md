# itch.io Page Copy

## Short Description

Progressive top-down extraction shooter with sector unlock events, dynamic threat-based extraction, and live performance modes.

## Full Description

Neon Descent is a fast-paced top-down extraction shooter where every run pushes deeper into increasingly dangerous sectors.

- Fight through evolving enemy pressure.
- Unlock sectors and survive breach events.
- Enter extraction under escalating threat.
- Tune your run with balance presets:
  - Arcade
  - Standard
  - Hardcore
- Use performance modes for smooth play across desktop and mobile.

## Controls

- Move: WASD / Arrow keys
- Ability: Space
- Pause: Esc
- Perf overlay: F3
- Cycle balance preset: F4

## Known Issues

- Very high-density fights can reduce visual clarity.
- Balance may continue to be tuned based on feedback.

# GitHub Release Description (v0.1.0-mvp)

Neon Descent `v0.1.0-mvp` is the first stable MVP release.

### Added
- Progressive sectors and gate breach events
- Biome progression flow
- Threat-based extraction escalation (LOW/HIGH/CRITICAL)
- Balance presets (`arcade`, `standard`, `hardcore`)
- Mobile perf modes (`auto`, `on`, `off`)
- Dynamic perf governor and in-run perf overlay

### Improved
- Anti-kite combat pressure
- Run pacing toward consistent 10-15 minute sessions
- Entity update efficiency and runtime stability

### Notes
- This MVP emphasizes core loop quality over broad content volume.
- Feedback is welcome for balance and readability polish.
